# BetterRest
#### 수면에 관한 csv파일과 Xcode의 CoreMl(기계 학습)을 이용하여 사용자의 숙면 시간을 추천 해주는 앱입니다.
### 추천을 위한 사용자에게 주어지는 질문은 총 3가지로 구성 되어있다.
1. 언제 일어나길 원하는지
2. 원하는 수면 시간
3. 하루 카페인 섭취량
